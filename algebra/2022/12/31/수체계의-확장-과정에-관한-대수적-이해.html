수체계의 확장 과정에 관한 대수적 이해

by Pragmo

December 31, 2022
in Algebra

개인적으로 생각하는 수와 연산의 기원

모든 학문의 발전은 현실세계에서 발생하는 문제를 해결하기 위해 고민하면서부터 시작한다

물고기를 일렬로 세운 모습을 본따 물고기 개수만큼 작대기를 그어 표기하는 원시 표기법(1진법)이 만들어져 사용된다

세기 편하게 개량된다

개량된 표기를 보니 저 1덩어리를 좀 줄여서 표기하고 싶다. 이름을 붙인다(추상화)

이후 다른 수들에도 이름을 붙인다(고도의 추상화)

‘없다’라는 상태에 이름을 붙인다(0의 발명)

기수법이 만들어져 사용된다

작대기 그은걸 모으는데서부터 시작된게 덧셈이다

곱셈은 원래는 ‘~가 몇개 있다’라는 말을 줄인 표기법이었으나

수에도 적용하기 시작하며 연산으로 발전하게 된다

페아노 공리계(변형)

‘사람을 죽이면 안된다’ 또는 ‘도둑질 하면 안된다’와 같은 세상의 규칙들은 너무나도 당연한 것이지만, 그것을 법이라는 시스템을 통해 체계적으로 관리하듯이

수학에서도 위에서 직관적으로 만들어낸 ‘자연수’, ‘덧셈’, ‘곱셈’을 수학적인 언어로서 체계적으로 정리할 필요가 있다

페아노 공리계는 그러한 필요에 따라 범자연수 N 0 (0을 포함하는 자연수)와 덧셈, 곱셈을 정의하는 방법들 중 하나이다

초창기 페아노 공리계는 자연수 집합에 0을 포함하지 않았으나, 0을 포함하는 것이 더 편리하기 때문에 후대의 수학자들이 0을 포함하도록 변형시킨 버전을 만들어냈다

현재 우리가 쓰고 있는 수체계에 관한 헌법(!)같은 존재로서 여기서 나오는 명제는 증명없이 사실로 받아들여야 한다

자연수의 성질

  1. 0은 자연수이다.
  2. 임의의 자연수 $n$에 대해, ‘따름수’ $S(n)$은 자연수이다. ex) $n = 0, S(n) = 1, S(S(n)) = 2$
  3. 임의의 자연수 $n$에 대해, $S(n) \neq 0$. 즉, 따름수가 0인 자연수는 존재하지 않는다.
  4. 임의의 자연수 $m$과 $n$에 대해, $S(m) = S(n)$이면 $m = n$
  5. $0 \in K$이고 임의의 자연수 $n$에 대해, $n \in K$이면, $S(n) \in K$이다. 이때, $K$는 모든 자연수를 포함한다.

자연수 사이의 덧셈

  1. 임의의 자연수 $a$에 대해 $a + 0 = a$
  2. 임의의 자연수 $a, b$에 대해 $a + S(b) = S(a + b)$

이들 공리로부터 수학적 귀납법을 활용해 다음 법칙이 유도된다

  1. 교환법칙 : $x + y = y + x$
  2. 결합법칙 : $(x + y) + z = x + (y + z)$
  3. 소거법칙 : $x + z = y + z \Rightarrow x = y$

자연수 사이의 곱셈

  1. 임의의 자연수 $a$에 대해 $a \times 0 = 0$
  2. 임의의 자연수 $a, b$에 대해 $a \times S(b) = a + (a \times b)$

이들 공리로부터 수학적 귀납법을 활용해 다음 법칙이 유도된다

  1. 교환법칙 : $x \times y = y \times x$
  2. 결합법칙 : $(x \times y) \times z = x \times (y \times z)$
  3. 소거법칙 : $x \times z = y \times z \Rightarrow x = y(z \neq 0)$
  4. 분배법칙 : $x \times (y + z) = x \times y + x \times z$

이로써 자연수란 무엇인가?와 자연수 범위에서의 덧셈과 곱셈이 정의되었다

$\mathbb{N}_0 \Rightarrow \mathbb{Z}$ 확장

자연수 범위에는 없지만

임의의 자연수 $n$에 대해

$n + x = 0$

이 되는 수 $x$가 있다고 가정한다

덧셈의 역원을 통해 확장한 형태이다

정수 범위의 공리

페아노 공리계에서 정의한 덧셈은 자연수 범위를 벗어나지 않으므로 위의 $n + x$ 덧셈 연산을 페아노 공리계를 활용해 설명할 수 없다

따라서 저 $x$가 있다고 가정한것 부터가 덧셈을 새롭게 정의한 것이라고 봐야할듯

간단히 말해서 ‘임의의 자연수와 더해서 0이 되는 수가 존재한다’는 공리가 하나 추가된 것이다

연산

정수 범위에서도 교환, 결합, 소거, 분배법칙이 성립한다고 가정한다

자연수 a, b, c에 대해

연산 - 덧셈

역원의 정의, 교환법칙, 결합법칙, 소거법칙을 활용한 증명

$(-a) + 0 + a = (-a) + a$

$\therefore (-a) + 0 = (-a)$부연설명1

교환, 결합법칙을 활용한 증명

$(-a) + (-b) = (a + b) - (a + b) + (-a) + (-b) = a + (-a) + b + (-b) - (a + b)$

$\therefore (-a) + (-b) = -(a + b)$

연산 - 곱셈

역원의 정의, 교환법칙, 분배법칙을 활용한 증명

$a + (-a) = 0$

$0 \times (a + (-a)) = 0 \times 0$

$\therefore (-a) \times 0 = 0$

역원의 정의, 분배법칙, 소거법칙을 활용한 증명

$a + (-a) = a + (-a)$

$1 \times (a + (-a)) = a + (-a)$

$a + 1 \times (-a) = a + (-a)$

$\therefore 1 \times (-a) = (-a)$부연설명2

역원의 정의, 교환법칙, 분배법칙을 활용한 증명

$(1 - 1) \times a = a + (-a)$

$a + (-1) \times a = a + (-a)$

$\therefore (-1) \times a = (-a)$부연설명3

역원의 정의, 분배법칙, 교환법칙, 결합법칙을 활용한 증명

$1 + (-1) = 0$

$(-1) \times 1 + (-1) \times (-1) = 0$

$\therefore (-1) \times (-1) = 1$

크기비교1

연산의 결과가 처음보다 작아지므로 $n + x = 0 \le n$

$\therefore x \le 0$

새로 발견한 수는 0보다 작거나 같음을 알아냈다

크기비교2

임의의 자연수 $n_1, n_2$에 대하여 그 역원을 $x_1, x_2$라고 하자

이때 $n_1 < n_2$이다

그렇다면

① $0 < n_2 - n_1$이다

역원의 정의에 따라

$n_1 + x_1 = n_2 + x_2 = 0$이다

② $n_2 - n_1 = x_1 - x_2$이다

①, ②에서 $0 < x_1 - x_2$이다

$\therefore x_2 < x_1$

즉 큰 수의 역원($x_2$)이 작은 수의 역원($x_1$)보다 작음을 알아냈다

$\mathbb{Z} \Rightarrow \mathbb(Q)$ 확장

정수 범위에는 없지만

임의의 정수 $z$(일단은 0까지도 포함해 생각해보자)에 대해서

$z \times x = 1$

이 되는 수 $x$가 있다고 가정한다

곱셈의 역원을 통해 확장한 형태이다

유리수 범위에서의 공리

페아노 공리계에서 정의한 곱셈은 자연수 범위를 벗어나지 않으므로 위의 $z \times x$ 곱셈 연산을 페아노 공리계를 활용해 설명할 수 없다

따라서 정수에서도 그랬듯이 저 $x$가 있다고 가정한것 부터가 곱셈을 새롭게 정의한 것이라고 봐야할 것 같다

‘임의의 정수와 곱해서 1이 되는 수가 존재한다’는 공리가 하나 추가된 것이다

연산

유리수 범위에서도 교환법칙, 결합법칙, 분배법칙, 소거법칙이 성립한다고 가정한다

임의의 정수 a, b, c에 대해

연산 - 곱셈

역원의 정의, 교환법칙으로 증명

$a \times a^{-1} = 1$

$\therefore a^{-1} \times 0 = 0$부연설명4

역원의 정의, 교환법칙, 결합법칙, 소거법칙으로 증명

$a^{-1} \times 1 \times a = a^{-1} \times a$

$\therefore a^{-1} \times 1 = a^{-1}$부연설명5

역원의 정의, 교환법칙, 결합법칙으로 증명

$a^{-1} \times b^{-1} = (a \times b) \times (a \times b)^{-1} \times a^{-1} \times b^{-1} = a \times a^{-1} \times b \times b^{-1} \times (a \times b)^{-1}$

$\therefore a^{-1} \times b^{-1} = (a \times b)^{-1}$

연산 - 덧셈

분배법칙으로 증명

$1 + 0 = 1$

$\therefore a^{-1} + 0 = a^{-1}$

교환법칙, 분배법칙, 결합법칙으로 증명

$a^{-1} + b^{-1} = b \times b^{-1} \times a^{-1} + a \times a^{-1} \times b^{-1}$

$\therefore a^{-1} + b^{-1} = (a + b) \times (a \times b)^{-1}$

0으로 나눈다는 모순

몇줄 위에있는 연산 - 곱셈 항목에서 보았듯이 모든 정수 $z$(0도 포함)에 대하여

$0 \times z^{-1} = 0$임이 확인되었다

즉 $0 \times 0^{-1} = 0$이다

한편, 0의 역원은 $0 \times 0^{-1} = 1$을 만족시키는 $0^{-1}$이다

0의 역원이 존재한다고 정해놨더니 $0 = 1$이 되는 결과가 생긴다

이를 두 연립방정식의 해 $0^{-1}$이 없다고 해석할 수도 있고

또는 기존 수체계의 붕괴를 막기 위해 0의 역원은 정의하지 않는다고 볼 수도 있을 것 같다

나눗셈이란 곱셈을 기반으로 한, 곱셈의 역연산으로서 정의된 연산이기 떄문에 곱셈으로 바꿀 수 없다면 나눌 수도 없다

0의 역원이 존재하지 않으므로 0으로 나눌 수 없다

크기비교1

새로 만든 수 x의 크기를 비교하기 위해 2가지 경우로 나누어 생각한다

i) $1 \le z$

$z \times x = 1 \le z$

① $x \le 1$

곱셈의 결과가 양수이므로

$0 < 1 = z \times x$

② $0 < x$

$\therefore 0 < x \le 1$

정수가 양수인 경우 그 역원은 0 초과, 1 이하임을 발견했다

ii) $z \le -1$

$-z \ge 1 = z \times x$

$-1 \le x$

곱셈의 결과가 양수이므로

$0 < 1 = z \times x$

$x < 0$

$\therefore -1 \le x < 0$

정수가 음수인 경우 그 역원은 -1 이상, 0 미만임을 발견했다

크기비교2

다음엔 두 수의 관계를 살펴보자

임의의 정수 $z_1, z_2$에 대하여 그 역원을 $x_1, x_2$라고 하자

$z_1 < z_2$이다

3가지 경우로 나눈다

i) $1 \le z$

$z_1x_1 = z_2x_2$

① $x_1 = \frac{z_2}{z_1}x_2$

$1 < \frac{z_2}{z_1}$

② $x_2 < \frac{z_2}{z_1}x_2$

①, ②에서 $x_2 < x_1 = \frac{z_2}{z_1}x_2$

$\therefore x_2 < x_1$

두 정수가 모두 양수인 경우에서 큰 수의 역원이 작은 수의 역원보다 작음을 발견했다

ii) $z \le -1$

$z_1x_1 = z_2x_2$

① $x_1 = \frac{z_2}{z_1}x_2$

$\frac{z_2}{z_1} < 1$

② $x_2 < \frac{z_2}{z_1}x_2$

①, ②에서 $x_2 < x_1 = \frac{z_2}{z_1}x_2$

두 정수가 모두 음수인 경우에서 큰 수의 역원이 작은 수의 역원보다 작음을 발견했다

$\therefore x_2 < x_1$

iii) $z_1 \le -1, 1 \le z_2$

$x < 0, 0 < x_2$

$\therefore x_1 < x_2$

두 정수중 큰 쪽이 양수이고 작은 쪽이 음수인 경우에서 작은 수의 역원이 큰 수의 역원보다 작음을 발견했다

정리

수체계를 확장한다는 것은 새로운 수를 발견할 뿐만 아니라 제대로 된 연산의 정의와 크기비교의 규칙을 만드는 것을 포함하는 일이다

즉 우리가 $10 \times 2.2$를 곱셈이 처음 만들어졌던 시기처럼 ‘-10이 2.2개 있다’로 해석하지 않고 -22라고 계산을 수행할 수 있는 것은 자연수 범위에서부터 지속적으로 확장된 수의 체계가 유리수 범위에서의 사칙연산을 잘 정의해주고 있기 때문이다

기수법과 더불어 이러한 연산을 가능하게한 수학 연구의 중요성을 느끼며 정리를 마무리한다.
